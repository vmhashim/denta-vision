DENTA VISION

Welcome to DENTA VISION! This project allows you to upload a tooth image to estimate the age based on dental features.

The application is developed using Streamlit and Python, and it's designed to process and analyze tooth images to provide an estimated age.

To run the application, please clone the repository and follow the instructions provided in the README.md file.
